## Architecture notes

- All tier surfaces follow the same pattern: helper lib → controls component → wired into `[surface].tsx`
- Every helper uses `AsyncSqliteLike` interface for testability with `bun:sqlite`
- Mutation results use `{ ok: true } | { ok: false; reason; message }` union types
- Surface controls are conditionally rendered by the dispatcher pattern (`if (surface === 'mobile.x') return <XControls />`)
- Entitlement gating is centralized in `canUseMobileSurfaceFromCache` — Tier A always works, paid surfaces fail closed after 7-day grace
